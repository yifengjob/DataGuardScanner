## 🎨 设计特点

### CSS 变量体系
```css
/* 浅色主题 */
--bg-color: #fff
--text-color: #333
--menu-bg: #f0f0f0
...

/* 深色主题 */
--bg-color: #141414
--text-color: #e8e8e8
--menu-bg: #1f1f1f
...
```

### 主题切换流程
1. 用户点击主题按钮
2. 循环切换：light → dark → system → light
3. applyTheme() 设置 data-theme 属性
4. CSS 变量自动应用新颜色
5. localStorage 保存选择
6. 界面即时更新（无需刷新）
